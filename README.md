# 🏡 Airbnb Price Predictor

A simple web application that predicts Airbnb listing prices based on user inputs.

## 🚀 Features
- Predict price based on property details
- Clean and modern UI
- Built using Flask

## 🛠 Tech Stack
- Python
- Flask
- HTML/CSS

## 📂 Project Structure
app.py
model.py
templates/
static/


## ▶️ How to Run

```bash
conda create -n airbnb python=3.10 -y
conda activate airbnb
pip install flask pandas scikit-learn
python app.py
