# 🏡 Airbnb Price Predictor

A simple web app that predicts Airbnb prices based on user inputs like bedrooms, bathrooms, amenities, and more.

---

## 🚀 Features
- Predict price instantly
- Clean and modern UI
- Built using Flask

---

## 🛠 Tech Stack
- Python
- Flask
- HTML / CSS

---

## 📂 Project Structure
```

airbnb_app/
├── app.py
├── model.py
├── static/
├── templates/
│   └── index.html

````

---

## ▶️ Run Locally

```bash
git clone https://github.com/Prerna-Karle/Airbnb-price-predictor.git
cd Airbnb-price-predictor

conda create -n airbnb python=3.10 -y
conda activate airbnb

pip install flask pandas scikit-learn

python app.py
````

Open in browser:

```
http://127.0.0.1:5000
```
